
# 🍳 Kitchen Chaos

Kitchen Chaos is a fast-paced cooking simulation game built with Unity.
Manage orders, cook dishes, and survive the chaos of a busy kitchen!

## 🎮 Gameplay Features

- 🔥 Time-based order system
- 🍔 Ingredient preparation & cooking mechanics
- 🧑‍🍳 Player movement & interaction system
- 🏆 Score & combo system
- ⏳ Increasing difficulty over time

## 🛠 Built With

- Unity (2022+ recommended)
- C#
- TextMeshPro
- Unity Input System

## 🚀 How to Play

1. Move using WASD / Arrow Keys
2. Interact using E / Space
3. Prepare ingredients
4. Assemble dishes
5. Deliver before time runs out!

## 📂 Project Structure

- Scenes/
- Animations/
- UI/
